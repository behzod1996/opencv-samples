# Camera Calibration with AR

This repository provides tools and resources for calibrating a camera and applying simple Augmented Reality (AR) techniques using OpenCV. 

## Tech Stack

- Python 3.7 or later
- OpenCV 4.5 or later
- NumPy
- SciPy
